Instructions

  • Select a "project" for the experiment to live in. The project determines the value of the category field in each post related to the experiment.
  • Select an experiment name (e.g., "foo"). All notes or posts for the experiment should carry a category of the format experiment:experiment-NAME.
  • Add a section to the experiments.html page, with an iterator over the posts for your experiment category.
The end result is that your experiment's posts will show up as a group on the main experiments page, they'll show up by clicking on the tag itself in the tag cloud listing, and the experiment's posts will show up as part of the overall project's listing.
